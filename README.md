# 🚗 JT Car Rentals v3.0

**Professional car rental management with React, Tailwind CSS, PDF generation & mobile support**

## ✨ Features
- ✅ React 18 with Babel (JSX support)
- ✅ Tailwind CSS styling
- ✅ Smooth scrolling
- ✅ Mobile responsive
- ✅ NO authentication (direct access)
- ✅ SQLite database
- ✅ Full CRUD operations
- ✅ Auto-calculations
- ✅ PDF generation with all details
- ✅ Customer fields: Phone, Address, NIC
- ✅ Real-time search
- ✅ Analytics dashboard
- ✅ Network access for mobile

## 🚀 Quick Start

```bash
npm install
npm run init-db
npm start
```

Open: `http://localhost:5000`

## 📱 Mobile Access
From phone on same WiFi:
```
http://<YOUR_IP>:5000
```

Get IP: `ipconfig` (Windows) or `ifconfig` (Mac/Linux)

## 📋 Fields
- Client Name, Phone, Address, NIC
- Vehicle Type
- Rental Dates (auto-calculates days)
- Mileage (auto-calculates used km)
- Charges & Deposits
- All in PDF

## 📄 PDF Includes
- Company name (JT Car Rentals)
- Complete customer info
- Vehicle details
- Charges breakdown
- Deposit information
- "Thank You for Choosing JT Car Rentals"
